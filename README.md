## Requirements
- Minecraft: 1.21.1
- Mod Loader: Fabric
- Fabric API: Required (unless stated otherwise in releases)

## Installation
1. Install Fabric Loader for Minecraft 1.21.1.
2. Install Fabric API.
3. Download the latest Clypton release from the Releases page.
4. Put the `.jar` into your `.minecraft/mods` folder.
5. Launch Minecraft.

## Compatibility
- Intended for DonutSMP (and similar server setups).
- Works best when used with a normal Fabric client setup (avoid mixing with incompatible mods).

## Configuration
Clypton supports configs:
- Location: `.minecraft/config/ClyptonClient`

## FAQ
